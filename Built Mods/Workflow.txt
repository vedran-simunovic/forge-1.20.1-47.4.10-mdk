Step 1: Build the Mod
--------------------
Run the Gradle build command from your project root:

    ./gradlew build

This will compile the mod and generate a JAR file.

---

Step 2: Locate the Built JAR
----------------------------
After the build completes, find the JAR in:

    simunovic-mod\build\libs

It will have a name like:

    simunovicmod-1.0-1.20.1.jar

---

Step 3: Test the Mod Locally
----------------------------
Copy the JAR file to your Minecraft mods folder to test:

    \Desktop\Minecraft 1.20.1\mods\

Launch Minecraft to make sure everything works as expected.

Tips & Best Practices
--------------------
- Use clear, descriptive names for versioned builds.
- Keep the `latest` folder up to date for easy access.
- Avoid committing temporary or local Gradle files (like `run-data/` or `build/`).
